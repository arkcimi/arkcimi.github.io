Regular Expression | 正则表达式

基本元素

字符

如 : a, 1……

  1. . : 任意字符(除了换行符)
  2. \d : 任意数字
  3. \w : 任意字母
  4. \s : 任意的空白符
  5. [] : 内部或 如:
    [123]: 1 or 2 or 3;
    [a-z]: a~z任一字符…… ;
  7. () : 块

数量

  • * : 0次以上
  • + : 1次以上
  • ? : 0 or 1次
  • {n} : n次
  • {n, } : n次以上
  • {n, m} : n~m次

语法

  • | : 或
  • ^ ：匹配字符串的开始
  • $ ：匹配字符串的结束
  • i : 不区分大小写
  • s : 可以换行

零宽断言

  • (?=任意字符) : (正前瞻)
  • (?<=任意字符) : (正后顾)
  • (?!任意字符) : (负前瞻)
  • (?<!任意字符) : (负后顾)

如 "\w*(?=1)" 匹配 'qw1er2', 得qw

贪婪和非贪婪

贪婪

如__”exp… \d{2,3}5 exp…”__ 会先匹配3个数字, 若下一个是5, 则继续匹配; 若不是5, 则会回溯, 匹配2个数字, 再检测是否匹配5. 若匹配失败, 整个表达式匹配失败.

非贪婪

在数量表达式后面加?
如 : *? , {n, m}?(先匹配n个, 再增加)
